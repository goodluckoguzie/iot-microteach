Learning Objectives

By the end of this session, students will be able to:

  1. Define the basic concept of Internet of Things (IoT).
  2. Identify the components of a simple sensor circuit.
  3. Construct a motion-detection logic using an Arduino microcontroller.

What is IoT?

The network of physical objects ("things") embedded with sensors and software to connect and exchange data.
