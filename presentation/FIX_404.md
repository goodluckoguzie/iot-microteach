## üêõ Troubleshooting

### Still Getting 404?

1. **Check repository name:**
   - Repository must be named exactly: `iot-microteach`
   - Case-sensitive!

2. **Verify files are in root:**
   - `index.html` should be in root (not in a subfolder)
   - Check: https://github.com/goodluckoguzie/iot-microteach/tree/main

3. **Check branch name:**
   - Should be `main` (not `master`)
   - If you have `master`, rename it: `git branch -M main`

4. **Clear browser cache:**
   - Hard refresh: `Ctrl+F5` (Windows) or `Cmd+Shift+R` (Mac)

5. **Wait longer:**
   - Sometimes takes 5-10 minutes for first deployment
